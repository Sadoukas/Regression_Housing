# Projet : Régression sur le Dataset Housing

Ce projet est un exercice d'entraînement au Machine Learning. L'objectif est d'utiliser le **dataset Housing** pour implémenter et comparer les **7 modèles de régression** présentés sur [GeeksforGeeks](https://www.geeksforgeeks.org/machine-learning/regression-in-machine-learning/).

## Modèles à implémenter :

1. Simple Linear Regression
2. Multiple Linear Regression
3. Polynomial Regression
4. Support Vector Regression (SVR)
5. Decision Tree Regression
6. Random Forest Regression
7. Ridge / Lasso Regression

## Installation

1. Activer l'environnement virtuel : \`source venv/Scripts/activate\`
2. Installer les dépendances : \`pip install -r requirements.txt\`
